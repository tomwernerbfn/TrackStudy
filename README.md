# TrackStudy 📚
*A terminal study tracker built in Python that helps you log study sessions, monitor your progress, and maintain a study streak.*

# Features

- Log study sessions with a name, topic, duration and extra notes
- Load and delete past sessions by date
- Track your total study hours and current streak

# Planned Features

- Per-subject hour breakdown
- Weekly goals system
- *Export to CSV*

# Requirements

Python 3.x

*How to Run*

git clone https://github.com/tomwernerbfn/TrackStudy.git
cd trackstudy
python trackstudy.py
